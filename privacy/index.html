Privacy

Privacy, the short version.

The long version would not say more. We watch banks, not visitors.

This website

liquilens.in is a static site. It sets no cookies, runs no advertising trackers and builds no visitor profiles. Aggregate, cookieless page counts via Cloudflare Web Analytics may be enabled; they identify nobody. The site is served by GitHub Pages behind Cloudflare, which produce standard server logs (IP address, user agent, timestamps) as any host does.

Email

If you write to mrinal091997@liquilens.in, your message and address are kept as ordinary correspondence, used to reply to you, and shared with nobody. Ask and the thread gets deleted.

The demo and the public API

The demo at demo.liquilens.in requires no signup and asks for no personal data. It contains public record evidence and a synthetic treasury book. The public API keeps standard access logs for security and rate limiting, with short retention, and nothing in it is personal data by design: the subjects are institutions, scored from their own public filings.

Pilots and the product

A pilot tenant's data is governed by the pilot agreement, which supersedes this page. The deployment we recommend to regulated lenders is self hosted, so your book never reaches our servers at all. We do not sell data. Not visitor data, not tenant data, not any data.
